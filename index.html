Tentang Saya

Saya adalah lulusan baru Sistem Informasi dari Jurusan Ilmu Komputer, Universitas Riau, dengan fokus dan passion di bidang pengembangan web. Dengan penguasaan teknologi seperti PHP, Laravel, HTML, dan Bootstrap 5, saya memiliki fondasi yang kuat untuk membangun aplikasi berbasis web. Sebagai seorang yang bersemangat dalam pembelajaran dan pengembangan diri, saya selalu antusias untuk menghadapi tantangan baru dan berkontribusi dalam tim. Tujuan karir saya adalah menjadi backend developer profesional yang dapat memberikan solusi teknologi yang efektif dan inovatif.

Proyek

Pekanbaru Rental Online

Projek ini dibuat untuk user yang ingin memesan rental secara online. Dengan mekanisme yang ada, user bisa melakukan pemesanan, memilih mobil, memilih mobil nya dijemput atau diantarkan, dan juga bisa memilih rental menggunakan supir dan tanpa supir. Projek ini dibangun menggunakan Laravel 11.

Balance Cairan

Projek ini dibuat untuk beberapa user, Perawat dan Pasien. Website ini nantinya digunakan untuk menghitung Balance Cairan dari pasien dengan menggunakan perhitungan tertentu dari berapa jumlah cairan Intake dan Output. Yang bisa menginputkan cairannya adalah Perawat dan Pasien. Projek ini dibangun menggunakan Laravel 11.

Web Flashsoft Indonesia

Projek ini dibuat untuk user yang ingin belajar di Flashsoft Indonesia. Jadi user yang masuk ke website ini bisa melihat beberapa informasi seperti paket belajar, bahasa pemrograman, dan harga nya. Projek ini dibangun menggunakan Laravel 11.
